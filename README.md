# vaik-rainbow-text-gif-generator

Generate rainbow text gif file


## Install

``` shell
pip install git+https://github.com/vaik-info/vaik-rainbow-text-gif-generator.git
```

## Usage

### Example

```shell
rainbow_text --text 7.77%
```
